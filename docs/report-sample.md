# 리포트 샘플 (v1.0)

## 요약
- GGUF 파일 파싱 과정에서 세그멘테이션 폴트 발생

## 재현 단계
1. 도커 컨테이너 실행
2. 동일 입력 파일로 3회 재현
3. 크래시 로그/스택 트레이스 확인

## 환경
- 컨테이너 이미지: <IMAGE_TAG>
- 타깃 버전: llama.cpp b7921
- 커밋/빌드 해시: <GIT_SHA>

## 크래시 로그
```
<CRASH_LOG>
```

## 스택 트레이스 (상위 3프레임)
```
<FRAME_1>
<FRAME_2>
<FRAME_3>
```

## 입력 파일 해시
- sha256: <INPUT_SHA256>

## 재현 커맨드
```
tool triage --input <PATH_TO_INPUT>
```
